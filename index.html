Rules of the takuzu:
  • Each box contains a binary value.In this case i use ▓ and ▄ , it can be 0 and 1)
  • Are not allowed more than two equal numbers or symbols each other. ▓ ▓
  • Each row or column contain an equal number of zeros(▓) and ones(▄).
  • Any row or column cannot be exactly equal to another.
